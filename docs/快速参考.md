# USART1 DMA环形缓冲接收 - 快速参考

## 📋 核心架构

```
USART1 (PA9/PA10, 115200bps)
    ↓ 硬件接收
DMA1 Channel5 (循环模式, 256字节双缓冲)
    ↓ 半满/全满中断
环形缓冲区 (1024字节)
    ↓ 5ms调度任务
uart_rx_task() 数据处理
```

---

## 🔌 硬件连接

| 功能 | CH32V引脚 | 外部设备 |
|------|----------|----------|
| USART1 Tx | PA9 | → USB转TTL Rx |
| USART1 Rx | PA10 | ← USB转TTL Tx |
| GND | GND | ↔ USB转TTL GND |

**波特率**：115200 | **数据位**：8 | **停止位**：1 | **校验**：无

## 🔍 调试技巧

### 1. 检查DMA是否接收数据

在`uart_rx_task()`中添加：

```c
static uint32_t last_print = 0;
uint32_t now = TIM_Get_MillisCounter();

if (now - last_print > 1000)  // 每秒打印一次
{
    last_print = now;
    printf("[DEBUG] Total RX: %lu bytes, Buffer: %u bytes\r\n",
           USART1_Get_RxCount(),
           ringbuffer_large_available(&g_usart1_ringbuf));
}
```

### 2. 查看DMA中断触发次数

在`ch32v30x_it.c:DMA1_Channel5_IRQHandler()`开头添加：

```c
static volatile uint32_t ht_count = 0, tc_count = 0;

if (DMA_GetITStatus(DMA1_IT_HT5) != RESET)
{
    ht_count++;  // 用调试器观察
    // ...
}
else if (DMA_GetITStatus(DMA1_IT_TC5) != RESET)
{
    tc_count++;  // 用调试器观察
    // ...
}
```

### 3. 启用统计任务

在`scheduler.c`中添加：

```c
#include "uart_test.h"

static task_t scheduler_task[] =
{
    {show_string, 10, 0},
    {uart_rx_task, 5, 0},
    {uart_debug_task, 1000, 0},  // ← 每秒打印统计
};
```

**输出示例**：

## 🐛 常见问题速查

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 无数据接收 | 硬件连接错误 | 检查Tx/Rx是否接反 |
| 数据乱码 | 波特率不匹配 | 确认双方都是115200 |
| 缓冲区溢出 | 处理速度慢 | 减少任务周期或增大缓冲区 |
| 丢失数据 | 中断优先级低 | 已设置为最高，检查是否禁用中断 |
| 编译错误 | 文件未添加 | 确认所有.c文件在编译列表 |
