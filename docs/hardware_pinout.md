## 5. 门禁控制 (舵机)

**控制方式**: PWM信号控制
**舵机类型**: 标准舵机 (0-180度)

| 功能 | 引脚 | 定时器通道 | GPIO配置 | 说明 |
|------|------|------------|----------|------|
| SERVO_PWM | PA6 | TIM3_CH1 | 复用推挽输出 | 舵机控制信号 |

- 频率: 50Hz (20ms周期)
- 脉宽范围: 0.5ms ~ 2.5ms
- 0度脉宽: 500μs
- 90度脉宽: 1500μs
- 180度脉宽: 2500μs
- 定时器: TIM3
- 定时器时钟: 96MHz
- 预分频器: 72-1 (1MHz计数频率)
- 自动重载值: 20000-1 (20ms周期)

**角度定义**:
- 锁定角度: 0度
- 解锁角度: 90度

**代码位置**: `User/door_control.h`, `User/door_control.c`

## 6. 指纹识别模块 (AS608)

**接口类型**: USART2串口通信
**模块型号**: AS608光学指纹识别模块
**默认波特率**: 57600

| 功能 | 引脚 | GPIO配置 | 说明 |
|------|------|----------|------|
| USART2_TX | PA2 | 复用推挽输出 | 发送数据到AS608 |
| USART2_RX | PA3 | 浮空输入 | 接收AS608数据 |
| AS608_STA | PB0 | 上拉输入 | 模块状态检测 |

**通信参数**:
- 波特率: 57600 (默认)
- 数据位: 8位
- 停止位: 1位
- 校验位: 无
- 模块地址: 0xFFFFFFFF (默认)

**功能特性**:
- 指纹容量: 最大300枚 (根据模块型号)
- 识别时间: <1秒
- 安全等级: 5级可调
- 支持指纹录入、验证、删除、清空

**硬件连接**:
```
AS608模块 → CH32V30x
VCC       → 3.3V/5V (根据模块要求)
GND       → GND
TXD       → PA3 (USART2_RX)
RXD       → PA2 (USART2_TX)
TOUCH     → PB0 (状态检测)
```

**代码位置**: `User/as608.h`, `User/as608.c`

## 7. 语音模块 (BY8301)

**接口类型**: UART4串口通信
**模块型号**: BY8301-16P
**默认波特率**: 9600

| 功能 | 引脚 | GPIO配置 | 说明 |
|------|------|----------|------|
| UART4_TX | PC10 | 复用推挽输出 | 发送控制命令到BY8301 |
| UART4_RX | PC11 | 浮空输入 | 接收BY8301状态信息 |

**通信参数**:
- 波特率: 9600 (固定)
- 数据位: 8位
- 停止位: 1位
- 校验位: 无

**功能特性**:
- 音频格式: MP3、WAV
- 音量等级: 0-30级可调
- 内置3W功放
- 支持语音播报、提示音播放

**硬件连接**:
```
BY8301模块 → CH32V30x
VCC (Pin6) → 3.3V
GND (Pin16) → GND
RX (Pin8)   → PC10 (UART4_TX)
TX (Pin7)   → PC11 (UART4_RX)
BUSY (Pin9) → (可选)
SPK1/SPK2   → 喇叭
```

**变更说明**:
- ⚠️ **V1.4.0更新**: 原计划使用USART3 (PB10/PB11)，但这两个引脚已被LCD占用
  - PB10 → LCD_RES (LCD复位信号)
  - PB11 → LCD_BLK (LCD背光控制)
- ✅ **新方案**: 改用UART4 (PC10/PC11)，引脚无冲突

**代码位置**: `User/by8301.h`, `User/by8301.c`

## 7.1 高频RFID读写器 (UART5)

**接口类型**: UART5 串口通信
**模块型号**: XH3650-A1/A2/B1/B2/C2系列
**工作频率**: 13.56MHz (ISO14443A)
**默认波特率**: 9600bps

### 引脚分配

| 功能 | 引脚 | GPIO配置 | 说明 |
|------|------|----------|------|
| UART5_TX | PC12 | 复用推挽输出 | 发送命令到RFID模块 |
| UART5_RX | PD2  | 浮空输入 | 接收RFID模块发送的数据 |

### 通信参数

| 参数 | 配置值 | 说明 |
|------|--------|------|
| 波特率 | 9600bps | 默认值，可通过命令设置为4800~115200 |
| 数据位 | 8位 | 标准配置 |
| 停止位 | 1位 | 标准配置 |
| 校验位 | 无 | 采用XOR校验和 |
| 流控 | 无 | 不使用硬件流控 |

### 硬件连接

```
RFID模块      CH32V307VCT6
━━━━━━━━━━━━━━━━━━━━━━━━━━
VCC (5V)  ──→ 5V电源 (独立供电)
GND       ──→ GND
TXD       ──→ PD2  (UART5_RX)  ← RFID发送，单片机接收
RXD       ←── PC12 (UART5_TX)  ← RFID接收，单片机发送
```

**⚠️ 重要说明：**
- RFID模块需要**5V供电**（模块内部有3.3V转换）
- TXD/RXD为**3.3V TTL电平**，可直接连接CH32V30x
- **不要将TXD/RXD接反**，否则无法通信
- 建议在VCC引脚并联**100μF+0.1μF滤波电容**
- 如走线较长(>20cm)，建议串联**22Ω电阻**防止反射

### 工作模式

RFID读写器支持两种工作模式：

**命令模式（默认）：**
- 由MCU主动发送命令触发读卡
- 适用于按需读卡的场景
- 响应帧头：0x02

**自动模式：**
- 卡片靠近自动上传数据
- 适用于快速识别场景
- 响应帧头：0x04
- 可配置：只上传卡号、只上传块数据、上传卡号+块数据

### 功能特性

- ✅ 读取卡号（UID）
- ✅ 读写数据块（16字节/块）
- ✅ 钱包功能（初始化、充值、扣款、查询）
- ✅ 扇区密码管理
- ✅ 工作模式切换（命令/自动）
- ✅ 蜂鸣器控制
- ✅ 波特率设置
- ✅ 固件版本查询

### 读卡距离

| 卡片类型 | 典型距离 | 最大距离 |
|---------|---------|---------|
| Mifare S50 | 3-5cm | 6cm |
| Mifare S70 | 2-4cm | 5cm |
| FM11RF08 | 3-5cm | 6cm |

**影响因素：**
- 天线质量
- 电源电压（建议5V±0.25V）
- 周围金属物体
- 卡片质量

### 电气规格

| 参数 | 最小值 | 典型值 | 最大值 | 单位 |
|------|--------|--------|--------|------|
| 供电电压 | 4.75 | 5.0 | 5.25 | V |
| 待机电流 | - | 50 | 100 | mA |
| 读卡电流 | - | 150 | 200 | mA |
| IO输出电平（高） | 2.4 | 3.3 | 3.6 | V |
| IO输出电平（低） | 0 | - | 0.4 | V |

**代码位置**: `User/rfid_reader.h`, `User/rfid_reader.c`
**API文档**: `docs/RFID_API_Reference.md`
**协议手册**: `高频RFID读写器系列协议手册V1.3.pdf`

## 11. 注意事项

### 11.1 定时器资源分配

✅ **已解决**: 定时器资源已合理分配:
- **TIM2**: 系统定时器 (1ms时基)
- **TIM3**: 舵机PWM控制 (50Hz)

### 11.2 电源要求

- **舵机电源**: 需要独立5V电源供电，电流需求通常为500mA-1A
- **AS608指纹模块**: 3.3V或5V供电（根据模块型号），工作电流<120mA
- **LCD背光**: 通过PB11控制，注意电流限制
- **矩阵键盘**: 扫描时注意行列切换延时

### 11.3 引脚复用注意事项

- PA2/PA3: USART2专用于AS608通信，不能用于其他功能
- PA6: TIM3_CH1专用于舵机PWM，不能用于其他功能
- PB13/PB15: SPI2引脚，不能用于其他GPIO功能
- PC10/PC11: UART4专用于BY8301通信，不能用于其他功能
- PC12/PD2: UART5专用于RFID通信，不能用于其他功能

### 11.4 硬件连接

**舵机接线**:
```
舵机信号线(黄/白) → PA6 (TIM3_CH1)
舵机VCC(红)       → 5V独立电源
舵机GND(黑/棕)    → GND (与主控共地)
```

**AS608指纹模块接线**:
```
AS608模块 → CH32V30x
VCC       → 3.3V/5V (根据模块要求)
GND       → GND
TXD       → PA3 (USART2_RX)
RXD       → PA2 (USART2_TX)
TOUCH     → PB0 (状态检测)
```

**BY8301语音模块接线**:
```
BY8301模块 → CH32V30x
VCC (Pin6)  → 3.3V
GND (Pin16) → GND
RX (Pin8)   → PC10 (UART4_TX)
TX (Pin7)   → PC11 (UART4_RX)
BUSY (Pin9) → (可选)
SPK1/SPK2   → 喇叭
```
