# LCD菜单系统使用指南

## 快速开始

### 1. 初始化菜单

```cpp
#include "menu_task.h"

// 在主程序中调用
void main() {
    menu_init();  // 初始化LCD和菜单系统
}
```

### 2. 定义菜单变量

```cpp
// 定义需要显示和修改的变量
static int temperature = 25;
static bool fan_enable = false;
static int brightness = 80;
```

### 3. 创建菜单项

```cpp
// 创建菜单项（标题、类型、变量指针、下一项、子菜单）
static MenuItem item1 = {"温度:", ITEM_INT, &temperature, NULL, NULL};
static MenuItem item2 = {"风扇:", ITEM_BOOL, &fan_enable, NULL, NULL};
static MenuItem item3 = {"亮度:", ITEM_INT, &brightness, NULL, NULL};
```

### 4. 添加到菜单

```cpp
menu->addItem(&item1);
menu->addItem(&item2);
menu->addItem(&item3);
menu->draw();  // 显示菜单
```

### 5. 处理按键

```cpp
// 在定时任务中调用（20ms周期）
void menu_input_task(void) {
    uint8_t key = read_key();  // 读取按键

    if (key != KEY_NONE) {
        menu->handleKey(key);  // 处理按键并自动刷新
    }
}
```

## 按键定义

```cpp
#define KEY_NONE   0  // 无按键
#define KEY_UP     1  // 上键
#define KEY_DOWN   2  // 下键
#define KEY_OK     3  // 确认键
#define KEY_CANCEL 4  // 取消键
```

## 按键操作

| 模式 | 按键 | 功能 |
|------|------|------|
| 浏览模式 | UP/DOWN | 切换选项 |
| 浏览模式 | OK | 进入编辑 |
| 编辑模式 | UP/DOWN | 修改数值 |
| 编辑模式 | OK | 切换布尔值 |
| 编辑模式 | CANCEL | 退出编辑 |

## 菜单项类型

```cpp
ITEM_INT   // 整数类型（可用UP/DOWN调整）
ITEM_BOOL  // 布尔类型（用OK切换ON/OFF）
ITEM_LINK  // 链接类型（预留，用于子菜单）
```

## 完整示例

```cpp
#include "menu_task.h"
#include "SimpleMenu.h"
#include "CH32_ST7735.h"

// 全局变量
static CH32_ST7735* tft = NULL;
static SimpleMenu* menu = NULL;

// 菜单变量
static int speed = 100;
static bool motor_on = false;

// 菜单项
static MenuItem item1 = {"速度:", ITEM_INT, &speed, NULL, NULL};
static MenuItem item2 = {"电机:", ITEM_BOOL, &motor_on, NULL, NULL};

void setup() {
    // 创建LCD对象
    tft = new CH32_ST7735();
    tft->setWidthHeight(128, 160);
    tft->begin();

    // 创建菜单对象
    menu = new SimpleMenu(*tft);
    menu->init();

    // 添加菜单项
    menu->addItem(&item1);
    menu->addItem(&item2);

    // 显示菜单
    menu->draw();
}

void loop() {
    // 读取按键（需要根据硬件实现）
    uint8_t key = KEY_NONE;

    if (GPIO_ReadInputDataBit(GPIOA, GPIO_Pin_0)) {
        key = KEY_UP;
    } else if (GPIO_ReadInputDataBit(GPIOA, GPIO_Pin_1)) {
        key = KEY_DOWN;
    } else if (GPIO_ReadInputDataBit(GPIOA, GPIO_Pin_2)) {
        key = KEY_OK;
    } else if (GPIO_ReadInputDataBit(GPIOA, GPIO_Pin_3)) {
        key = KEY_CANCEL;
    }

    // 处理按键
    if (key != KEY_NONE) {
        menu->handleKey(key);
        Delay_Ms(200);  // 防抖
    }
}
```

## 注意事项

1. **按键防抖**: 建议添加 200ms 延时避免重复触发
2. **变量作用域**: 菜单变量必须是 `static` 或全局变量
3. **指针有效性**: MenuItem 结构体必须在菜单使用期间保持有效
4. **内存管理**: LCD 和菜单对象使用 `new` 创建，注意内存

## 接口参考

### SimpleMenu 类

```cpp
class SimpleMenu {
public:
    SimpleMenu(Adafruit_GFX& gfx);  // 构造函数
    void init();                     // 初始化
    void draw();                     // 绘制菜单
    void handleKey(uint8_t key);     // 处理按键
    void addItem(MenuItem* item);    // 添加菜单项
};
```

### MenuItem 结构体

```cpp
typedef struct MenuItem {
    const char* title;        // 标题
    uint8_t type;            // 类型
    void* value;             // 变量指针
    struct MenuItem* next;   // 下一项
    struct MenuItem* child;  // 子菜单
} MenuItem;
```
